# MicroPython - Cupcake Machine
Order four types of cupcakes.


## Installation
```bash
git clone https://github.com/microbit-katie/Cupcake-Machine.git
```

## Run 
```bash
python main.py
```

## Run Tests
```bash
python -m unittest tests/test_funcs.py
```
